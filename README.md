# Лабораторная работа №6  
## Сборка многомодульных программ. Численные методы

## 📌 Описание

Необходимо реализовать программу для вычисления площади плоской фигуры, ограниченной тремя кривыми:

- y = f1(x)  
- y = f2(x)  
- y = f3(x)  

Вычисления выполняются с заданной точностью ε.

---

## 💡 Примеры

📎 Полная постановка задачи находится в файле `task.md`.

📎 Варианты заданий находятся в файле `variants.md`.  
📎 Примеры отчета в tex, а также простой примеры многомодульной программы находятся в папке `examples/`.

---

## 🧠 Задачи

### 1. Поиск точек пересечения

Найти корни уравнений вида:

f(x) = g(x)

с точностью ε₁, используя численные методы:

- метод деления отрезка пополам  
- метод хорд (секущих)  
- метод Ньютона  
- комбинированный метод  

Метод выбирается на этапе компиляции через `#define`.

---

### 2. Вычисление интегралов

Вычислить определённые интегралы с точностью ε₂:

- метод прямоугольников  
- метод трапеций  
- метод Симпсона  

---

### 3. Вычисление площади

Площадь фигуры представляется как комбинация определённых интегралов.

---

## ⚙️ Требования

- Функции `f1`, `f2`, `f3` реализуются на ассемблере (NASM32)  
- Остальной код реализуется на языке C  
- Используется соглашение вызова `cdecl`  
- Сборка выполняется с помощью `make`  

---

## 🧩 Основные функции

### root

```c
double root(
    double (*f)(double),
    double (*g)(double),
    double a,
    double b,
    double eps
);
```

### integral

```c
double integral(
    double (*f)(double),
    double a,
    double b,
    double eps
);
```

---

## 🖥️ Аргументы командной строки

- `--help` — список параметров  
- `--roots` — точки пересечения  
- `--iterations` — количество итераций  
- `--test-root` — тестирование root  
- `--test-integral` — тестирование integral
